# snakemake profile for ATLAS

tested on slurm, easily adaptable for lsf and other cluster systems.

How to use:

1. copy the folder 'cluster/atlas' in '$HOME/.config/snakemake'
2. change parameters and paths in config.yaml and cluster_spec.yaml
3. run atlas with the command option ' --profile atlas '

more informations:
https://snakemake.readthedocs.io/en/stable/executable.html#profiles
https://github.com/snakemake-profiles
